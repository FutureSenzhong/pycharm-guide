10.3 【数据库操作 03】查询结果格式化
====================================

|image0|

表格形式
--------

查询结果默认显示以表格的形式打印，你可以很方便的复制到 Excel 上

|image1|

JSON格式
--------

如果你需要将结果展示成 JSON 的格式，点击切换到 ``Tree``

|image2|

CSV 格式
--------

如果你需要将结果展示成 CSV 的格式，点击切换到 ``Text``
